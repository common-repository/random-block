== Description ==

Create a random layout using this plugin

= Feedback =
* I am open for your suggestions and feedback - Thank you for using or trying my plugin!

== Installation ==

1. Upload the entire `random` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

= Minimum Requirements =

* WordPress 2.1 or higher

= Manual Installation =

1. Unzip the files and upload the folder into your plugins folder (/wp-content/plugins/) overwriting older versions if they exist
2. Activate the plugin in your WordPress admin area.

== Frequently Asked Questions ==

= Does this plugin work with newest WP version and also older versions? =
Yes, this plugin works really fine with WordPress 5.2.1!
It also works great with the oldest version as well as the latest version also.

= In which WordPress version this Plugin is compatible? =
It is compatible from 2.1 to 5.2.1 WordPress version.
